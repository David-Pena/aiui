## Quick Start

```bash
# Install with npm
npm install @ai-ui/core

# Or with yarn
yarn add @ai-ui/core

# Or with pnpm
pnpm add @ai-ui/core
```

## Usage

```vue
<script setup>
import { UButton } from '@ai-ui/core'
</script>

<template>
  <UButton variant="primary">
    Click me!
  </UButton>
</template>
```
